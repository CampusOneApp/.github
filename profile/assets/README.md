Built for schools, colleges, universities and training institutes,

## Company

CampusOne is building the next generation digital operating system
for educational institutions.

Instead of using multiple disconnected software solutions for attendance,
academics, communication, administration and reporting,
CampusOne connects everything through one integrated experience.

Our goal is simple:

> Build software that allows institutions to focus on education,
> while CampusOne manages technology.

# Platform Modules

<p align="center">

Every institution is different.

CampusOne provides modular architecture so organizations can enable only the
features they need.

</p>

<br>

| Module | Description |
|--------|-------------|
| <img src="./assets/icons/student.svg" width="18"> **Student Management** | Complete student lifecycle management from admission to graduation. |
| <img src="./assets/icons/faculty.svg" width="18"> **Faculty Portal** | Teaching, attendance, schedules and academic workflows. |
| <img src="./assets/icons/attendance.svg" width="18"> **Attendance System** | QR, biometric and manual attendance with analytics. |
| <img src="./assets/icons/communication.svg" width="18"> **Communication Hub** | Announcements, notices and institutional messaging. |
